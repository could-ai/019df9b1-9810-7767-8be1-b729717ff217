# ARCHIVE ACADEMY

ARCHIVE ACADEMY একটি সোশ্যাল নেটওয়ার্কিং প্ল্যাটফর্ম যেখানে ব্যবহারকারীরা সাইন-আপ এবং লগইন করতে পারবেন, নিজেদের প্রোফাইল দেখতে পারবেন, এবং হোম ফিডে পোস্ট করতে পারবেন। 

## ফিচারসমূহ (Features)
- **Authentication:** ব্যবহারকারীদের জন্য নিরাপদ লগইন এবং সাইন-আপ অপশন।
- **Home Feed:** সবার পোস্ট একসাথে দেখার ব্যবস্থা।
- **Post Modal:** নতুন পোস্ট তৈরি করার সুবিধা।
- **Profile Page:** ব্যবহারকারীর নিজের তথ্য ও ছবি দেখার পেজ।

## টেকনোলজি স্ট্যাক (Tech Stack)
- **Frontend:** Flutter
- **Backend/Database:** Node.js, MongoDB Atlas (Future Scope), Local Mock State (Current Scope)
- **Tools:** VS Code, Postman

## রান করার নিয়ম (How to Run)
1. প্রথমে রিপোজিটরি ক্লোন করুন।
2. টার্মিনালে `flutter pub get` রান করুন।
3. প্রজেক্ট রান করার জন্য `flutter run` কমান্ড ব্যবহার করুন।

## CouldAI সম্পর্কে
এই অ্যাপটি [CouldAI](https://could.ai) ব্যবহার করে তৈরি করা হয়েছে। CouldAI একটি এআই অ্যাপ বিল্ডার যা ক্রসিং-প্ল্যাটফর্ম (iOS, Android, Web, এবং Desktop) অ্যাপ তৈরির জন্য ব্যবহৃত হয়। এর মাধ্যমে সাধারণ প্রম্পট ব্যবহার করে প্রোডাকশন-রেডি অ্যাপ তৈরি করা সম্ভব।
